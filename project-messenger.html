메신저 협업툴

팀원 간 실시간 소통과 원활한 업무 관리를 목적으로 개발된 커스텀 협업 메신저입니다. 채널 기반의 분리된 채팅 환경을 지원하며 파일 공유 및 알림 기능을 제공합니다.

GitHub 저장소

주요 기술 스택

Vue.js

반응형 UI 설계 및 컴포넌트 아키텍처를 기반으로 한 확장성 높은 프론트엔드 구축.

Spring Boot

안정성 있는 백엔드 API 서비스 제공 및 효율적인 비즈니스 로직 처리.

WebSockets

소켓 통신을 이용해 대기 시간 없는 실시간 양방향 채팅 및 알림 동기화 기능 구현.
