단백질 체크

운동인을 위한 하루 단백질 및 영양소 섭취량 기록 애플리케이션입니다. 직관적인 UI를 통해 식단을 손쉽게 기록하고, 목표 달성률을 시각화하여 건강 관리에 도움을 줍니다.

GitHub 저장소

주요 기술 스택

React Native

크로스 플랫폼 모바일 애플리케이션 개발. iOS 및 Android 동시 지원.

Firebase

사용자 인증(Authentication) 및 백엔드 서비스 연동으로 데이터 실시간 동기화 개발.

Redux

전역 상태 관리를 통해 복잡한 컴포넌트 간 데이터 상태를 효율적으로 관리.
